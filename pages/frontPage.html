Get Started

  • Easy to Use

    Our application is easy to use, all you need to do is start our crawler, gather data then create a statistic using the data you gathered.

  • Gather data

    You can easily gather data using our crawler, it works with multiple social platforms such as Facebook, Twitter, Reddit, Tumblr and so many more.
    As long as a platform has an API, you can configure our crawler for it.

  • Real-time statistics

    We use Chart.JS for our statistics, for beautiful and responsive charts.
    You're guaranteed to find a chart suited for your desired statistic, plus it will be updated in real-time so you won't miss anything
